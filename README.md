# PrepCheck

**AI-powered Work Readiness Screener for Students and Career Starters**

PrepCheck is a self-assessment platform designed to help students evaluate their readiness for job placements. By analyzing resumes and form inputs, PrepCheck provides instant feedback, personalized improvement suggestions, and a readiness score to guide users toward career success.

---

## Features (v1.0)

- Upload a resume (PDF/DOCX) or fill out a form
- Automatic parsing and scoring based on predefined metrics
- Readiness score (0â€“100) with evidence-based justification
- Top 3 personalized suggestions to improve readiness
- AI-powered feedback with schema-bound output for analytics

---

## AI Capabilities

- Extract structured data: skills, experience, education, certifications
- Match user profile to predefined career streams
- Generate concise rationale for readiness score
- Recommend resources based on identified gaps
